Portal público de información del edificio

Desde este portal, la Junta Directiva centraliza información importante para los propietarios: minutas, procedimientos de pago, presupuesto de mantenimiento, porcentajes, sanciones, formatos de reserva y preguntas frecuentes. Todo el contenido está pensado para una consulta rápida desde celular o computadora.

Uso recomendado del portal

Recomendaciones de consulta para propietarios y residentes.

  1. Revisar primero la sección FAQ antes de realizar una consulta en el grupo.
  2. Verificar las minutas y acuerdos cuando se necesite confirmar una decisión adoptada.
  3. Consultar mantenimiento y porcentajes para validar información económica del edificio.
  4. Usar los enlaces oficiales del portal para pagos, reservas y documentos.
