Notificaciones

Esta página está organizada para separar visualmente los anuncios emitidos por la inmobiliaria y por la administración.

Inmobiliaria

Carpeta sugerida para publicar PNG oficiales.

  • Abrir carpeta de inmobiliaria
  • Usar esta sección para garantías, postventa, entregas, cronogramas y comunicados formales.
  • Se recomienda nombrar los archivos por fecha para mantener el orden.

Administración

Carpeta sugerida para avisos operativos del edificio.

  • Abrir carpeta de administración
  • Usar esta sección para avisos de agua, energía, mantenimiento, horarios y coordinaciones internas.
  • Las publicaciones también pueden acompañarse con fecha, resumen y enlace complementario.
